========
CSS GRID
========

# vh
- viewport height
- tamanho do browser/navegador inteiro

# display: grid
- colocar no pai

# grid-template-columns
- no pai
- quantidade de colunas do template
- cada uma com seu tamanho
- ex: 25% 25% 25% 25%; 
- 100% dividido por 4 colunas

# grid-template-rows
- no pai
- quantidade de linhas do template
- cada uma com seu tamanho
ex: 33.33% 33.33% 33.33%; 
- 100% dividido por 3 linhas

# grid-column-start / end
- coluna de início e a de fim (número)
- sempre mandar ir uma a mais para ele poder preencher ela
- atalho: grid-column: 1 / 4; (início e fim)

# grid-row-start / end
- linha de início e a de fim (número)
- sempre mandar ir uma a mais para ele poder preencher ela
- atalho: row-column: 1 / 3; (início e fim)

# gap
- adiciona espaçamento de uma coluna / linha para outra
